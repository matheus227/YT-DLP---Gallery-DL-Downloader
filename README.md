# YT-DLP---Gallery-DL-Downloader
## ! importante ! **Código corrigido Por uma I.A**
Um código em python que aproveita as ferramentas yt-dlp e gallery-dl

No windows 10 ou 11 a instação do yt-dlp e gallery-dl é feita através de uma ferramenta nativa: "winget", ela baixa os programas e instala até as dependências invisíveis (como o FFmpeg, que o yt-dlp usa para juntar áudio e vídeo) de forma totalmente automática.

 1-execute o terminal(prompt de comando)
 
 2- instalação do yt-dlp e gallery-dl:
   yt-dlp => "winget install yt-dlp.yt-dlp"
   gallery-dl => "winget install mikf.gallery-dl"

  Pronto, yt-dlp e gallery-dl estão instalados no seu computador.

  Para que o windows atualize suas rotas e reconheça corretamente as ferramentas, feche o terminal e abra novamente.
